Paquetería en Kubernetes con Helm

Event: #TechParty2019 Date: 09/14/2019

Summary

Declaring Kubernetes configuration explicitly can become tedious and difficult to maintain as applications grow.

This talk introduced Helm as a way to package, install and manage Kubernetes applications without losing the core capabilities of Kubernetes. It covered what Helm is, how it is used, and a practical analysis of its advantages and trade offs.
