Kubernetes como Plataforma de Desarrollo

Event: HackMadrid %27 Date: 06/30/2021

Event page

Summary

Kubernetes has become the de facto platform for running cloud native applications and powering modern CI and CD pipelines. However, despite its capabilities, it has historically been underused as a development platform.

In this talk, we explore how Kubernetes can provide consistent, reproducible development environments using open source tools. We will discuss the challenges of local development, how cloud based development environments improve collaboration and onboarding, and demonstrate practical solutions that bring developers closer to production while simplifying the overall development workflow.
